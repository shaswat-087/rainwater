# Project Title
RAINWATER HARVESTING SYSTEM

## Overview
This project estimates harvestable rainwater based on rooftop area, rainfall, and roof type .

### Features
- Shows an user interface to input roof dimesions, roof surface type and location (district) 
- Calculates rooftop area
- Uses district rainfall data
- Estimates annual and daily water availability
